Privacy Policy for GitHub Reviews Bot

Last updated: April 14, 2026

GitHub Reviews Bot is a Chrome extension that adds one-click buttons on GitHub pull request pages to post review commands for supported bots.

Information We Collect

GitHub Reviews Bot does not collect personal information, account credentials, or browsing history.

The extension only stores your bot toggle preferences (enabled/disabled state) using chrome.storage.sync. This allows your preferences to sync across Chrome browsers where you are signed in.

How Information Is Used

  • To remember which review bot buttons you want to show or hide.
  • To render and trigger your selected review command on GitHub pull request pages.

Data Sharing and Sale

We do not sell, rent, or share your data with third parties.

GitHub Reviews Bot does not transmit extension data to external servers controlled by the developer.

Permissions

  • storage: used only to save your bot preferences.
  • GitHub pull request page access (https://github.com/*/pull/*): used only to inject and operate review buttons on PR pages.

Changes to This Policy

This policy may be updated from time to time. Any updates will be posted on this page with a revised date.

Contact

For questions or support, please open an issue at https://github.com/sabatesduran/github-reviews-bot-buttons/issues.
